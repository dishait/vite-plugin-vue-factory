# vite-plugin-vue-factory

vite的vue可视化工程管理插件

<br />

## Features 🦖

- pnpm 的
- `vitest` 测试
- 开箱即用的
- `typescript` 的

<br />
<br />

## Usage 🦕


### install

```shell
# 工作区安装
pnpm i

# 源码依赖安装
pnpm i -w
```

### init:info

```shell
pnpm init:info
```

### test

```shell
pnpm test

# or pnpm test:watch
```

### build

```shell
pnpm build
```

### coverage

```shell
pnpm coverage
```

### dev

```shell
pnpm dev
```

### publish

```shell
npm publish
```

### play

```shell
# 工作区 dev
pnpm play

# or pnpm play:open
# or pnpm play:host
# or pnpm play:build
# or pnpm play:preview
# or pnpm play:preview:open
# or pnpm play:preview:host
```

### release

```shell
pnpm release
```
